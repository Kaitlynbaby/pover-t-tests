The code runs without any problem on a computer with 2 Quad CPU and
8GB RAM, a GTX960 (4GB RAM) GPU and with Linux Mint 18.2 OS.

Install Requirements:
Python – version 3.6.4
scikit-learn - version 0.19.1
pandas  - version 0.22.0
numpy - version 1.14.0
lightgbm - version 2.0.11
keras - version 2.1.2
tensorflow  - version 1.3.0


Replicating the Submission:
1. Put the 12 data raw csv files in code/data/
2. Create directory code/predictions
3. Run run.py, which is located in /code/ 
4. The final submission file will be in code/predictions/final_submission_test.csv
